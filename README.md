# Toy Language Interpreter

A thread-safe toy language interpreter built with Java and a JavaFX graphical user interface. This educational tool allows users to visualize the internal workings of a language interpreter by executing scripts step-by-step and monitoring the internal memory structures in real-time. 💻🔍

## Features

* **🖥️ Interactive GUI:** A clean JavaFX interface to interact with the interpreter.
* **⏯️ Step-by-Step Execution:** Control the flow of the program using the "Run one step" button to see exactly how each statement affects the program state.
* **📊 Real-Time Visualization:** The interface dynamically displays the core components of the interpreter during execution:
    * **Execution Stack:** Displays the sequence of statements pending execution.
    * **Symbol Table:** Tracks initialized variables and their current values in the local scope.
    * **Heap:** Monitors dynamically allocated memory, showing memory addresses and their corresponding values.
    * **Out:** Displays the standard output (e.g., `print` statements) generated by the toy program.
    * **File Table:** Keeps track of open files and their identifiers.
* **🔀 Concurrency Support:** Displays the "Number of Program States" and "Program State IDs", allowing you to inspect different active threads.
* **🛡️ Thread-Safe:** The underlying architecture is thread-safe, ensuring accurate execution and state visualization even with concurrent program states.

## 🛠️ Technologies Used
* **Language:** Java
* **UI Framework:** JavaFX
